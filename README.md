# Ozon Tech
### app.js
Здесь находиться Api.
Чтобы передать значение, нужно использовать функции:


* `progress.setValue(n)` - для установления процентного соотношения длины загрузки;
* `progress.setHidden(true/false)` - для скрытия или показа блока загрузки;
* `progress.setAnimate(true/false)` - для включения анимации загрузки.
